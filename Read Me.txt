# superShopTest_v2.1.0
fully javascript based and without any database, just for practice. 

You can :
1. add new item and auto get the price of that item.
2. when quantity change then auto get the update price of the item.
3. update any item easily from the table. Fixed the update problem. 
4. display the total price in the alert box with better way such as date and time.   
5. get the new table for new customer. 
6. print facility, print final result in paper. 
7. if you have pdf creator install then final result will be print as a pdf file. 
8. confirmation system. 

*It has no any database such as sql and these data will no be stored. 
*Most of the elements add dynamically in here. 
*Fixing most of the problem. 
*Working to control by using keyboard.
